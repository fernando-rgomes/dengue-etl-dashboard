# ETL - Pipeline de Dados de Dengue no Brasil (2026)

Este projeto implementa um pipeline de **ETL (Extract, Transform, Load)** utilizando dados reais de casos de dengue no Brasil. Os dados foram obtidos do DATASUS e processados utilizando Python para limpeza, transformação e posterior armazenamento em um banco de dados PostgreSQL.

O objetivo do projeto é demonstrar habilidades práticas em **engenharia de dados, manipulação de dados e integração com bancos relacionais**, simulando um fluxo real de processamento de dados utilizado em projetos de análise e ciência de dados.

---

# Fonte dos dados

Os dados utilizados neste projeto foram obtidos do:

DATASUS — Departamento de Informática do Sistema Único de Saúde

Dataset utilizado:
Casos de dengue no Brasil (2026)

Portal oficial:
https://datasus.saude.gov.br/

O DATASUS disponibiliza dados epidemiológicos públicos utilizados para monitoramento de doenças e planejamento de políticas de saúde pública no Brasil.

---

# Objetivo do projeto

Construir um pipeline de dados que simule um fluxo real de processamento de dados epidemiológicos, incluindo:

- Extração de dados brutos
- Limpeza e tratamento
- Engenharia de atributos
- Armazenamento em banco de dados
- Preparação dos dados para análise
